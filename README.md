# PNU-analyzer-2

PNU(19자리)를 기반으로 건축물대장 정보를 조회하여
주택유형, 세대수, 주소, 면적 등을 자동으로 정리하는 Streamlit 기반 도구입니다.

---

## 🚀 주요 기능

* PNU 기반 건축물대장 API 조회
* 주용도명, 세대수, 건물정보 자동 정리
* PNU별 요약 + 건축물 상세 데이터 생성
* 엑셀 다운로드 기능 제공

---

## 📂 입력 데이터

* Excel(.xlsx)
* 필수 컬럼: `pnu` (19자리)

---

## 📊 출력 결과

엑셀 파일 (2개 시트)

1. PNU별_요약

   * 주소
   * 주용도명
   * 세대수 합계
   * 건물동수
   * 데이터생성일자

2. 건축물_상세

   * 건축물대장 표제부 상세 정보

---

## ⚙️ 실행 방법

```bash
pip install -r requirements.txt
streamlit run pnu_bldrgst_streamlit_app.py
```

## 🔑 API

* 공공데이터포털 건축물대장 API 사용
* 서비스키(Decoding) 필요
